# Stroke Dataset - Data Preprocessing

## Overview

This project performs data preprocessing on the Healthcare Stroke Prediction Dataset to prepare it for machine learning. The dataset is cleaned by handling missing values, removing duplicate records and outliers, correcting data types, encoding categorical variables, scaling numerical features, removing irrelevant columns, and handling skewness.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Output

A cleaned dataset (`Stroke_Dataset_Cleaned.csv`) ready for machine learning and data analysis.
